# DASHBOARD KPP SEMARANG

## REQUIREMENT

- NODE JS
- BUN

# HOW RUN THIS PROJECT ON LOCAL

Clone this repository by running this command:

```bash
git clone https://github.com/yudhaginongpratidina/kkp-semarang.git
cd kkp-semarang
```

For Development:

```bash
bun install
bun run dev
```

Copy the link below and open it in your browser:

```
http://localhost:5173
```
